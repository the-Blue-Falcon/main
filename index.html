TIDoS Framework

An advanced, comprehensive tool for web-penetration testing. Very useful during the reconnaissance or information gathering phase.

It is tool written in Python. It's really an advanced tool having lots of modules (presently 25) like dos attack, scanning, lookups and many more. It is still in beta phase and is upgrading its capabilities. It has achived 500+ (now near to 600) stars on github. Many of pentesters made tutorials about this tool, so you can easily get information you needed about it. You can download this tool directly from github, install it on any linux machine and start using it from today itself.

Brute Panel

Brute Panel is an advanced, modern admin login panel finder written in Python. It encorporates four modules, namely, bruteforcer (includes a huge wordlist), google hunter as well as dorker, and finally a port scanner.
