CSRF Probe

CSRF Probe is modern advanced scanner for hunting down Cross Site Request Forgery bugs in various web applications.

This tool has a wide range of awesome features. Amongst them all, this scanner features continuous crawling and scanning simultaneously with full support for both GET and POST type HTTP requests. Support for custom cookie values and generic headers is absolute. This tool has the capability to intelligently generate special crafted tokens for different parameter values. Another important frature is that this scanner has rare chances of false positives. It also generates pocs (proof of concepts) for both exploitable and not exploitable CSRFs It has a user-friendly interaction environment and everything is automated on demand..

Jolter

Jolter is a advanced multithreaded Network Stress and Denial of Service Audit Toolkit. A stress-testing suite made for analyzing the strength of your website against different kinds of denial of service attacks.

This stress testing suite includes 3 major modules, i.e. HTTP FLOOD, SYN FLOOD and UDP FLOOD which are the most commonly used in Denial of Service attacks. An awesome feature of this tools is that everything is multithreaded for high effectiveness and moreover, this tool spoofs requests with random IPs to prevent exposure. The HTTP FLOODER included in this is a Layer 7 flooder which can bring down bring down unpatched web servers within seconds. And like all of our other tools, it also has a user-friendly interaction environment.

Brute Panel

Brute Panel is modern advanced admin login path finder written in Python.

This tool includes a automated in-built bruteforcer + google dorker + port scanner which bruteforces for directories as well as paths. Has a big path list (near about 3000 paths) for accurate detection and prepares custom report at end of admin panel gathering. Supports php, asp, cgi, brf, js, jsp, cfm and html extensions with support for custom paths. Similar and with no exception, it also has a user-friendly interaction environment.
